# Hemophilia Emergency Planner

Deploy-ready Next.js prototype based on the CDC family emergency checklist for hemophilia families.

## Local run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload the contents of this folder.
3. In Vercel, click **Add New Project**.
4. Import the GitHub repository.
5. Keep the default Next.js settings.
6. Click **Deploy**.

## Notes

- This version is self-contained and only depends on `next`, `react`, `react-dom`, and `lucide-react`.
- No database or authentication is included.
- It is a prototype and not medical advice.
